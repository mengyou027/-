该资源包主要通过物品模型映射来改变模型，可以在不安装模组的情况下使用，并且以开源的方式添加手办的模型数量。

This resource pack primarily alters models through item model mapping, allowing for use without installing mods and enabling the addition of figurine models in an open-source manner.
